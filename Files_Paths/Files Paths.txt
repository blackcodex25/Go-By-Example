File Paths ใน Go คือการจัดการและการสร้างเส้นทางไฟล์ที่มี
ความเป็นพกพาระหว่างระบบปฏิบัติการต่างๆ (เช่น การใช้ / บน
Linux และ \ บน Windows) โดยใช้แพ็คเกจ filepath เพื่อ
ช่วยให้การจัดการเส้นทางไฟล์เป็นไปอย่างสะดวกและถูกต้อง

รายละเอียดการทำงานและฟังก์ชันหลัก:
1.Join: ฟังก์ชันนี้ใช้เพื่อสร้างเส้นทางไฟล์ที่ถูกต้องตาม
มาตรฐานของระบบปฏิบัติการโดยอัตโนมัติ โดยไม่ต้องกังวลว่าจะต้องใช้
เครื่องหมาย  / หรือ \ ให้ถูกต้องด้วยตัวเอง

2.Dir และ Base: ใช้สำหรับแยกเส้นทางไฟล์เป็นส่วนของ
directory และ filename หรือสามารถใช้ฟังก์ชัน Split เพื่อ
คืนทั้งสองส่วนพร้อมกัน

3.IsAbs: ใช้ตรวจสอบว่าเส้นทางไฟล์เป็นเส้นทางแบบสมบูรณ์
(absolute path) หรือไม่

4.Ext และ trimSuffix: ใช้จัดการส่วนขยายของไฟล์ เช่น
การแยกนามสกุลไฟล์ออกจากชื่อไฟล์

5.Rel: ใช้หาความสัมพันธ์ของเส้นทางไฟล์ระหว่าง base และ target
ถ้าไม่สามารถหาความสัมพันธ์ได้ จะคืนค่าข้อผิดพลาด (error)
