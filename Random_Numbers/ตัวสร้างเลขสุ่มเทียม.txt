ตัวสร้างเลขสุ่มเทียม (Pseudorandom Number Generator หรือ PRNG) เป็นเครื่องมือ
ที่ใช้สร้างลำดับของตัวเลขที่ดูเหมือนจะเป็นสุ่ม แต่จริงๆ แล้วสามารถคาดเดาได้ถ้าเรา 
รู้วิธีการสร้างหรือความรู้เบื้องต้นเกี่ยวกับสภาวะเริ่มต้นของมัน  ตัวเลขที่สร้างขึ้นไม่ใช่สุ่มจริงๆ
แต่ถูกสร้างขึ้นตามลำดับที่กำหนดไว้ล่วงหน้า

หลักการทำงานของ PRNG
1.Seed Value: ตัวเลขแรกที่ใช้ในการเริ่มต้นลำดับของตัวเลขสุ่ม
โดยทั่วไปจะเรียกว่า seed.Seed เป็นค่าเริ่มต้นที่ถูกกำหนดเพื่อให้ PRNG 
เริ่มต้นจากลำดับที่กำหนด 
2.Algorithm: PRNG ใช้สูตรทางคณิตศาสตร์หรือฟังก์ชันเพื่อคำนวณลำดับของ
ตัวเลขจากค่า seed ตัวอย่างเช่น อัลกอริธึมที่ใช้บ่อยคือ Linear Congruential Generator (LCG)
Mersenne Twister, หรือ Xorshift
3.Period: PRNG มีช่วงหรือระยะเวลาที่ลำดับของตัวเลขจะวนกลับไปที่ค่าเดิม 
ช่วงนี้เรียกว่า period ของ PRNG

ตัวอย่างการสร้าง PRNG ใน Go
ใน Go, ฟังก์ชัน math/rand ใช้สำหรับสร้างตัวเลขสุ่มเทียม
// ตั้งค่า seed ด้วยค่าเวลาเพื่อให้ลำดับสุ่มแตกต่างกันในแต่ละครั้งที่รันโปรแกรม
	rand.Seed(time.Now().UnixNano())

	// สร้างตัวเลขสุ่มระหว่าง 0 ถึง 99
	for i := 0; i < 5; i++ {
		fmt.Println(rand.Intn(100))
	}

การทำงานของโค้ด
1.Seed Initialization:
rand.Seed(time.Now().UnixNano()) ใช้ค่าเวลา (Unix timestamp)
เป็น seed ซึ่งจะเปลี่ยนทุกครั้งที่โปรแกรมรัน ทำให้ลำดับของตัวเลขสุ่ม
เปลี่ยนแปลงในแต่ละครั้ง

2.Random Number Generation:
rand.Intn(100) สร้างตัวเลขสุ่มที่อยู่ระหว่าง 0 ถึง 99

การใช้ PRNG ในการพัฒนา
1.การทดสอบ: ใช้ PRNG เพื่อสร้างข้อมูลทดสอบที่หลากหลายและมีความซ้ำซ้อนต่ำ
2.การจำลอง: ใช้ PRNG ในการจำลองสถานการณ์ต่างๆ เช่น การเล่นเกมหรือการจำลองทางคณิตศาสตร์
3.การป้องกัน: ในบางกรณีที่ต้องการความปลอดภัยสูง เช่น การเข้ารหัสลับเราอาจต้องใช้
crypto/rand ซึ่งใช้การสร้างตัวเลขสุ่มที่แท้จริง

ข้อจำกัดของ PRNG
Predictability: PRNG สามารถคาดเดาได้ถ้าคุณรู้ seed และอัลกอริธึมที่ใช้
Reproducibility: PRNG สามารถสร้างลำดับที่เหมือนกันได้ถ้าใช้ seed เดิม

การใช้ PRNG ในการพัฒนา
การใช้ PRNG เหมาะสำหรับงานที่ไม่ต้องการความปลอดภัยสูง และต้องการการสร้าง
ตัวเลขสุ่มที่คาดเดาได้อย่างง่ายดาย ในกรณีที่ต้องการความปลอดภัยที่สูงมาก เช่น
การเข้ารหัสลับ ควรใช้การสร้างตัวเลขสุ่มที่แท้จริงจากแพ็คเกจ crypto/rand
