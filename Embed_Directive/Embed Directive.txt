ใน Go 
//go:embed เป็นคำสั่งของคอมไพเลอร์ที่ช่วยให้เราสามารถ
ฝังไฟล์หรือโฟลเดอร์ต่างๆ เข้าไปใน binary ของโปรแกรมได้ในขณะที่ทำการคอมไพล์
ซึ่งทำให้เราสามารถเข้าถึงข้อมูลเหล่านั้นได้ในเวลารันไทม์โดยไม่ต้องพึ่งพาการโหลดจาก
ไฟล์ภายนอก

หลักการทำงานของ //go:embed:
1.การ Import  embed package:
เราจะต้องนำเข้า package embed เพื่อใช้ฟังก์ชันการฝังไฟล์ แต่หากเราไม่ใช้ identifiers 
ใดๆ จาก package นี้ เราสามารนำเข้าโดยใช้ blank import (_ "embed")

2.การฝังไฟล์เข้าไปในตัวแปร:
คำสั่ง //go:embed สามารถใช้ในการฝังไฟล์เข้ากับตัวแปรประเภท string, []byte, 
หรือ embed.FS
การฝังไฟล์จะเกิดขึ้นในขณะคอมไพล์ และข้อมูลที่ฝังจะถูกเก็บอยู่ใน binary ของโปรแกรม
ซึ่งหมายความว่าไฟล์ที่ถูกฝังจะกลายเป็นส่วนหนึ่งของโปรแกรมและสามารถเข้าถึงได้โดยตรง
ในเวลารันไทม์

3.รูปแบบการฝังไฟล์:
เราสามารถฝังไฟล์เดี่ยวๆ เข้าไปในตัวแปร string หรือ []byte
สามารถฝังหลายไฟล์หรือโฟลเดอร์โดยใช้ตัวแปร ประเภท embed.FS
ซึ่งทำงานเหมือนกับระบบไฟล์เสมือน (virtual file system) และสามารถใช้ wildcard
(*) เพื่อฝังไฟล์หลายๆ ไฟล์ได้พร้อมกัน
